Data Security

数据安全说明

最后更新：2026-05-15

跨境卖家的成本、利润、供应链和投放数据都属于敏感商业信息。SKUAuditPro 的产品设计原则是：先用最少数据完成利润判断，不要求店铺后台权限。

我们不要求的信息

  • 不要求店铺后台账号和密码。
  • 不要求广告账户、支付账户或平台 API 权限。
  • 不要求上传供应商合同、客户名单或完整财务流水。

建议提交的数据

  • SKU 名称或内部代号。
  • 售价、采购价、物流成本、平台费、佣金、广告成本、退货率。
  • 如涉及关税测算，可提供产品材质、用途、目的国和申报价值范围。

安全措施

  • 启用 HTTPS 加密访问。
  • 仅保存完成利润判断所需的必要数据。
  • 限制敏感操作访问，记录关键操作日志。
  • 按角色隔离客户数据和管理员权限。
  • 为导出文件、报告和订单信息设置访问控制。

我们的原则

SKUAuditPro 优先使用你主动提供的基础成本数据完成判断，不主动索取店铺账号、支付账户或广告账户权限。
